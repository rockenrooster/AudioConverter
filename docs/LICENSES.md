# Licenses

AudioConverter depends on FFmpeg and bundled FFmpeg runtime dependencies.

Release packages should include:

- App license.
- FFmpeg license text matching the selected build configuration.
- License text for bundled codec/runtime DLL dependencies.
- The FFmpeg configure line and source version.
- SHA-256 hashes for shipped DLLs.

Do not make legal claims from this file. Treat GPL/LGPL posture as a release-owner decision.
